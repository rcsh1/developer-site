如果您在生产环境中发布或更新您的 Cobo Portal Apps ，将自动触发一个工作流，需要 Cobo 的批准。一旦获得批准，您的应用将对 Cobo Portal 上的所有用户可用。您可以使用 `status` 命令检查 Cobo Portal Apps 的状态。

确保您从项目目录中运行命令，其中包含Manifest 文件。

## 用法
```shell
cobo app status
```

## 示例
```shell
cobo app status
```

此命令显示 Cobo Portal Apps 的 App ID 和当前状态。
