使用此命令通过快捷方式在您的默认网络浏览器中方便地打开 Cobo Portal 的各个页面。

<Note>使用某些快捷方式（如 `dashboard`、`wallets`、`custodial`、`mpc`、`scw` 和 `exchange`）需要登录和认证。如果您未登录，将被重定向到登录页面。</Note>

## 用法

```shell
cobo open <shortcut>
```

## 参数

`<shortcut>`：用于快速打开 Cobo Portal 页面的快捷方式。

可用的快捷方式包括：

- `portal`：Cobo Portal 的登录页面
- `dashboard`：仪表板页面
- `wallets`：钱包概览页面
- `custodial`：全托管钱包概览页面
- `mpc`：MPC 钱包概览页面
- `scw`：智能合约钱包概览页面
- `exchange`：交易所钱包概览页面
- `developer`：开发者控制台
- `apps`：Cobo Portal Apps概览页面
- `pricing`：定价方案页面
- `approval`：审批页面

## 示例

```shell
cobo open mpc
```

此命令打开 MPC 钱包概览页面。
