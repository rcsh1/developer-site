使用 `run` 命令在本地运行 WaaS App 。

## 用法
```shell
cobo app run [options]
```

## 选项
- `-p`, `--port <INTEGER>`： App 将监听传入请求的端口。如果不使用此选项， App 将运行在默认端口 `5000`。
- `-i`, `--iframe`：通过 iframe 加载 App 。

## 示例

```shell
`cobo app run --port 8080`
```

此命令启动 WaaS App 并在端口 8080 上监听传入请求。
