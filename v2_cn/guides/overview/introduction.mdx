Cobo 钱包即服务（WaaS）2.0 API 是 Cobo WaaS API 的最新版本。它使您能够通过强大且灵活的访问控制来访问 Cobo 的全套加密钱包技术。通过封装复杂的安全协议并简化区块链交互，该 API 使您能够专注于您的核心业务，而无需担心资产的安全性。WaaS 2.0 API 具有以下主要特点：

- 统一的 API，支持 Cobo 的[所有四种钱包类型](https://manuals.cobo.com/cn/portal/introduction#an-all-in-one-wallet-platform)
- 支持 80+ 条链和 3,000+ 代币
- 全面的 Webhook 事件
- 灵活的 MPC 钱包使用模式，包括[机构钱包](https://manuals.cobo.com/cn/portal/mpc-wallets/ocw/introduction)和[终端用户钱包](https://manuals.cobo.com/cn/portal/mpc-wallets/ucw/introduction)
- 对智能合约钱包（如 Safe\{Wallet\}）的程序化控制，提供精细的访问控制
- 在多个交易所之间无缝转移资金，包括 Binance、OKX、Bybit、Deribit 等

您可以将 WaaS 2.0 API 与 [Cobo Portal](https://manuals.cobo.com/cn/portal/introduction) 结合使用，后者是一个 Web 平台，具有以下开发者功能：

- [开发者控制台](https://manuals.cobo.com/cn/portal/developer-console/introduction)，您可以在这里监控 API 活动、管理 webhook 事件、查看日志等。
- [风控机制](https://manuals.cobo.com/cn/portal/risk-controls/introduction)，包括可自定义的交易风控规则、业务风控规则和用户角色及权限。
- [仪表盘](https://manuals.cobo.com/cn/portal/dashboard/introduction)，提供了用于资产管理的各类实时数据，包括总资产价值、资产分布、交易记录以及套餐的使用统计。

使用 WaaS 2.0 API，您可以：

- **为您的 App 赋能**：作为交易所、支付服务提供商、原生加密公司、场外交易商或零售平台，您可以将 Cobo 强大的钱包功能无缝集成到 App 中。通过这种集成，您可以提供专业级的数字资产管理服务，为用户创造更大价值，在市场中建立竞争优势。
- **构建** [**Cobo Portal Apps**](https://manuals.cobo.com/cn/portal/portal-apps/introduction),通过 WaaS 2.0 API，您还可以构建 Cobo Portal Apps 来显著扩展 Cobo Portal 的功能，如质押、交易和自动化金融操作。
