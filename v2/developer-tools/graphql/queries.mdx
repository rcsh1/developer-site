This section provides a list of available queries and their descriptions.

## addressConnection
Retrieve a paginated list of addresses.

Returns [AddressConnection](/v2/developer-tools/graphql/schema#addressconnection).

**Permissions**

| Resource  | Operation |
|-----------|-----------|
| wallet    | read      |

For more details about permissions, refer to [Permissions and wallet scopes](/v2/guides/overview/permissions-and-scopes).

**Arguments**

| Name       | Type      | Description |
|------------|----------|-------------|
| `walletId` | `String!` | The wallet ID. |
| `addresses` | `String` | A list of wallet addresses, separated by comma. For addresses requiring a memo, append the memo after the address using the `\|` separator (e.g., `address\|memo`). |
| `chainIds` | `[String]` | A list of chain IDs, which are the unique identifiers of blockchains. |

## chain

Returns [Chain](/v2/developer-tools/graphql/schema#chain).

**Permissions**

This query does not require any permission.

**Arguments**
| Name     | Type     | Description |
|---------|---------|-------------|
| `chainId` | `String!` | The chain ID, which is the unique identifier of a blockchain. |
